## Цель
Изучить синтаксис JavaScript: переменные, типы данных, операторы, условия, функции. Сравнить с C#.

## Ключевые отличия: JS vs C#

| Аспект | C# | JavaScript |
|--------|-----|------------|
| Типизация | Статическая | Динамическая |
| Компиляция | Да (IL) | Нет (JIT/интерпретация) |
| Точка входа | `Main()` | Отсутствует |
| Объявление переменной | `int x = 5;` | `let x = 5;` |
| Вывод в консоль | `Console.WriteLine()` | `console.log()` |
| Сравнение | `==` по значению | `===` строгое, `==` с приведением |
| Null-значения | `null` | `null`, `undefined` |
| ООП | Классическое | Прототипное + классы |

##  Выводы
1. **Гибкость**: JS позволяет менять типы переменных динамически.
2. **Простота**: Меньше синтаксического шума, быстрее писать.
3. **Риски**: Ошибки типов проявляются только при выполнении.
4. **Безопасность**: C# отлавливает больше ошибок на этапе компиляции.
